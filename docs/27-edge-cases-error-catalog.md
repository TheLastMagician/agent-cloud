# 模块二十七：边界场景与完整错误目录

> 本文档枚举系统中所有已知的边界场景、异常情况、以及对应的处理策略。
> 实现时每个场景都必须有明确的处理逻辑。

---

## 1. Agent 行为边界

### 1.1 死循环检测

```
场景: Agent 反复执行相同的工具调用且结果不变

检测方法:
├── 连续 3 次相同工具调用 + 相同参数 → 警告
├── 连续 5 次 → 强制中断, 要求换策略
└── 连续 10 次 → 终止会话

实现:
  track recent_tool_calls[]
  if last 3 calls identical:
    inject system_reminder: "你似乎在重复相同操作, 请尝试不同方案"
  if last 5 calls identical:
    force end_turn
```

### 1.2 Token 预算耗尽

```
场景: 上下文窗口接近 200K 限制

处理:
├── 70% 占用 → 开始压缩旧轮次
├── 85% 占用 → 积极截断工具输出
├── 95% 占用 → 摘要模式 (只保留最近 3 轮)
└── 99% 占用 → 强制生成最终回复

风险: 压缩导致上下文丢失 → Agent 忘记之前的决策
缓解: 保留关键决策点的摘要
```

### 1.3 无限子代理嵌套

```
场景: 子代理内部再启动子代理

限制:
├── 子代理不能启动子代理 (设计约束)
├── 子代理的工具集不包含 Task 工具 (除 generalPurpose)
└── generalPurpose 理论上可以, 但通过 prompt 限制

深度限制: 最大 1 层嵌套
```

## 2. 工具执行边界

### 2.1 文件系统边界

```
场景                          │ 处理
──────────────────────────────┼─────────────────────────
文件不存在 (Read)             │ 返回错误消息, Agent 搜索正确路径
文件权限不足 (Read/Write)     │ 返回错误消息, Agent 可能 chmod
文件过大 (Read, >2MB)         │ 建议使用 offset/limit
二进制文件 (Read)             │ 返回错误或 base64 (图片)
文件路径含特殊字符             │ 需要转义处理
符号链接循环                   │ 检测并报错
磁盘空间不足 (Write)          │ 返回错误, Agent 清理空间
StrReplace old_string 不唯一  │ 返回出现次数, 要求更多上下文
StrReplace old_string 不存在  │ 返回错误, Agent 重新读取文件
并发写入同一文件               │ 串行化 (工具内部锁)
```

### 2.2 Shell 执行边界

```
场景                          │ 处理
──────────────────────────────┼─────────────────────────
命令超时 (默认 30s)            │ 杀死进程, 返回 timeout 错误
命令产出巨量输出                │ 截断到 MAX_OUTPUT_SIZE
交互式命令 (等待输入)           │ 超时后杀死, 建议非交互替代
后台进程已存在                  │ Agent 应检查 terminals/ 避免重复
PID 不存在 (想要杀死)           │ 返回错误
端口被占用                     │ 返回错误, Agent 找到并杀死旧进程
环境变量未设置                  │ 命令可能失败, Agent 分析输出
命令不存在 (command not found)  │ 返回错误, Agent 安装或换命令
SIGKILL vs SIGTERM             │ 先 SIGTERM, 等 5s, 再 SIGKILL
```

### 2.3 搜索工具边界

```
场景                          │ 处理
──────────────────────────────┼─────────────────────────
Grep 无匹配结果                │ 返回空结果, Agent 换关键词
Grep 匹配过多 (>5000 行)      │ 截断 + 提示 truncated
Glob 无匹配                   │ 返回 "0 files found"
搜索 node_modules 等大目录     │ ripgrep 默认跳过 .gitignore
正则表达式语法错误              │ 返回 regex 错误
搜索超时                       │ 截断已有结果返回
```

## 3. 子代理边界

### 3.1 computerUse 边界

```
场景                          │ 处理
──────────────────────────────┼─────────────────────────
页面未加载完成                  │ 智能等待 (截屏比较) 最多 10s
元素坐标计算错误                │ 重新截屏分析, 调整坐标
弹出对话框阻挡                  │ 截屏识别并关闭
浏览器崩溃                     │ 重启 Chrome, 导航回原 URL
多标签页混乱                   │ 截屏确认当前标签
下载对话框                     │ 按 Escape 关闭或选择保存
JavaScript alert/confirm      │ 按 Enter 确认
Cookie 同意横幅                │ 寻找关闭按钮点击
CAPTCHA                       │ 无法处理 → 报告给用户
2FA 验证                       │ 需要用户提供 OTP seed 或手动登录
页面重定向循环                  │ 检测并报错
SSL 证书错误                   │ Chrome 启动参数跳过
```

### 3.2 debug 子代理边界

```
场景                          │ 处理
──────────────────────────────┼─────────────────────────
Bug 无法复现                   │ 报告无法复现, 建议代码审查
插桩代码引入新 Bug              │ 清理插桩, 重新开始
```

## 4. VM 与基础设施边界

### 4.1 VM 生命周期边界

```
场景                          │ 处理
──────────────────────────────┼─────────────────────────
VM 创建超时 (>60s)             │ 重试 1 次, 仍失败 → 报错
VM 启动后立即崩溃               │ 检查 rootfs, 重建
VM 内存不足 (OOM)              │ 增加配额 or 清理进程
VM 磁盘满                     │ 清理 /tmp, Docker 缓存
VM 网络中断                    │ 等待 30s 恢复, 否则重建
VM 快照保存失败                 │ 重试 1 次, 仍失败 → 继续 (不阻塞)
快照恢复失败                   │ 回退到冷启动 (全新创建)
快照过期/损坏                   │ 冷启动 + 重新环境设置
Update script 失败             │ 记录警告, 继续 (尽力策略)
Update script 超时 (>300s)     │ 杀死, 记录警告, 继续
```

### 4.2 Docker-in-Docker 边界

```
场景                          │ 处理
──────────────────────────────┼─────────────────────────
Docker daemon 未启动            │ 自动启动 dockerd
Docker pull 超时               │ 重试, 配置镜像加速
Docker 磁盘空间不足             │ docker system prune
Docker Compose 依赖启动失败    │ 检查日志, 重试
Docker 网络冲突                │ 清理旧网络, 重建
Docker 版本不兼容              │ 固定版本 (28.x)
```

## 9. 计费与配额边界

```
场景                          │ 处理
──────────────────────────────┼─────────────────────────
月初配额重置时正在运行会话      │ 允许完成, 不中断
付费失败 (信用卡过期)           │ 宽限期 7 天, 降级到 Free
降级套餐后超出新限额            │ 允许完成已有会话, 新会话受限
退款后的使用量追溯              │ 不追溯, 计为已消耗
Token 计数不精确 (跨模型)       │ 使用 tiktoken 精确计数
免费用户尝试超额                │ 硬性拒绝, 提示升级
企业合同到期                   │ 宽限期 30 天
```

## 10. 数据一致性边界

```
场景                          │ 处理
──────────────────────────────┼─────────────────────────
Redis 与 PostgreSQL 数据不一致  │ PostgreSQL 为准, Redis 做缓存
事件丢失 (Redis crash)         │ 重放机制 + 幂等消费
快照与代码库版本不匹配          │ git pull 后运行 update_script
并发创建同名 Secret             │ UNIQUE 约束 + 乐观锁
会话状态更新冲突                │ 乐观锁 (updated_at check)
产物上传中断                   │ 分块上传 + 断点续传
审计日志写入延迟                │ 异步写入, 最终一致
```
